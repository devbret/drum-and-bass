# Drum And Bass Visualized

An interactive, zoomable knowledge graph for visualizing conceptual nodes, enabling users to explore, drag, preview and inspect interconnected ideas within electronic music culture.

## Overview

Renders an interactive, zoomable knowledge graph which visualizes relationships between conceptual nodes and their associated tags, dynamically generating both idea–tag and tag–tag connections based on shared metadata. Users can drag nodes, explore structural proximity, preview descriptions via hover tooltips for selected node types and click idea nodes to populate a detail panel. Effectively transforming a typed dataset into an exploratory, spatial interface for understanding linkages and the broader conceptual architecture of electronic music culture.
